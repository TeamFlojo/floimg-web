# Glossary

User-facing terminology for floimg.com. Ensures consistency across marketing, docs, and UI copy.

---

## Product Names

| Term                    | Usage                    | Context                             |
| ----------------------- | ------------------------ | ----------------------------------- |
| **FloImg**              | The product/project name | Marketing, headlines                |
| **FloImg Studio**       | Visual workflow builder  | Product pages, feature descriptions |
| **FloImg Studio Cloud** | Managed cloud service    | Pricing page, cloud features        |

**Never use**: "Floimg", "FloIMG", "the FloImg", "Studio Cloud" (without "FloImg")

---

## Key Concepts

Use these terms consistently in docs and marketing copy:

| Term              | Definition                                                 |
| ----------------- | ---------------------------------------------------------- |
| **Workflow**      | A sequence of image operations (generate, transform, save) |
| **Pipeline**      | Synonym for workflow; use "workflow" in user-facing copy   |
| **Generator**     | Creates images (AI, charts, diagrams, QR codes)            |
| **Transform**     | Modifies images (resize, crop, blur, composite)            |
| **Deterministic** | Guaranteed identical output for identical input            |

---

## User Segments

How we refer to different user types:

| Segment                    | Description                               |
| -------------------------- | ----------------------------------------- |
| **Developers**             | Using SDK/CLI for programmatic workflows  |
| **Creative Professionals** | Using Studio for visual workflow building |
| **Teams**                  | Organizations using FloImg Studio Cloud   |
